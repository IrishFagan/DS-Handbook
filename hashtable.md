# Hash Table

A hash table is a structure that uses a hash function to convert a key into a value.

# Operations

A Hash Table supports the following operations:

* **Search**: 
  * O(n), linear time.

* **Insertion**: 
  * O(n), linear time.

* **Deletion**: 
  * O(n), linear time.

# Examples

* **creation**:

~~~
HT = HashTable()
~~~

* **access**:

~~~
print(HT[30])
~~~

* **search**:

~~~
HT.get(dog)
~~~

* **insert**:

~~~
HT[7] = "cat"
~~~

* **delete**:

~~~
del HT[7]
~~~
